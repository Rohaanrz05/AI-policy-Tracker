# 🛡️ AI Policy Tracker

## 📌 Project Overview

AI Policy Tracker is an intelligent policy classification and monitoring system designed to analyze, categorize, and track policy-related documents using Machine Learning techniques. The system helps organizations, researchers, and policymakers efficiently manage large volumes of policy data by automating classification and providing actionable insights.

---

## 🚀 Features

- 📄 Upload and analyze policy documents
- 🤖 Machine Learning-based policy classification
- 📊 Interactive dashboard with visual analytics
- 🔍 Real-time policy tracking and monitoring
- 📈 Data visualization using charts and graphs
- 🏷️ Automatic policy category prediction
- 💾 Model persistence using Joblib
- 🌐 User-friendly Streamlit web interface

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/AI-Policy-Tracker.git
cd AI-Policy-Tracker
```

### 2. Create Virtual Environment

```bash
python -m venv venv
```

### 3. Activate Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux/Mac

```bash
source venv/bin/activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Application

Start the Streamlit server:

```bash
streamlit run app.py
```

The application will launch in your browser at:

```text
http://localhost:8501
```
